内容简介
   
  本书介绍了Visual Basic和SQL Server 2000的必备知识，并完整地介绍了几个使用Visual Basic开发SQL Server 2000数据库应用程序的实例，包括学生档案管理系统、软件项目开发管理系统、旅游资源及线路管理系统和企业库存管理系统，既可以作为独立的系统运行，也可以作为应用软件的一个模块，具有很强的实用性和可移植性。本书最后还给出了程序优化和发布方法。
本书适合SQL Server 2000数据库管理员和应用程序开发人员，以及对相关技术感兴趣的读者阅读。阅读者中人要具备基本的Visual Basic编程知识，无需有任何经验，就可以按照书中的实例开发数据应用程序。
